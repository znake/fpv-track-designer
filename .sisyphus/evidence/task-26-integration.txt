- Store → Scene: `currentTrack` provides gates array, `config` provides fieldSize
- Store → Sidebar: All UI components consume store state directly via `useAppStore`
- Gate selection: Click in 3D → `useGateSelection` hook → `selectGate` in store → `GateAdjustment` UI updates
- Gate adjustment: UI controls → `updateGate`/`moveGate`/`rotateGate` in store → Scene re-renders
- Shuffle: `TrackControls` → `setTrack` in store → Scene shows new track
